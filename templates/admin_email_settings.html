Setup Guide
  1. Go to Google Cloud Console
  2. Create a new project (e.g. "FreshPlate Email")
  3. Enable the Gmail API in APIs & Services → Library
  4. Configure OAuth consent screen (External)
    • Add your Gmail as a Test User
  5. Create OAuth 2.0 credentials (Web Application)
    • Redirect URI: http://localhost:5000/admin/email-oauth/callback
  6. Download the JSON → rename to gmail_credentials.json
  7. Place it in your project root directory
  8. Click "Connect Gmail" above
Security Note

Only the gmail.send scope is requested. The app cannot read your emails.

Email Triggers

Customers receive emails on these status changes:

Received Order confirmation
Ready Ready for delivery
Delivered Delivery confirmation

Logged and In Preparation do not trigger emails (internal workflow steps).
